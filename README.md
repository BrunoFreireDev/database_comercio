# Modelando DBs

Repositório dedicado aos meus estudos de modelagem de banco de dados.

Aqui desenvolvo modelos conceituais, lógicos e físicos para diferentes tipos de negócios, com o objetivo de praticar estruturação de dados e boas práticas de design de banco de dados.

## Estrutura do repositório

Cada pasta representa um cenário de negócio diferente.

- Comercio → Modelagem de banco para operações comerciais
- Livraria → Estrutura de dados para gestão de livros, vendas e clientes
- Mecanica → Sistema para controle de ordens de serviço e peças
- Varejo → Operações de venda ao consumidor
- Atacado → Controle de vendas em grande volume para empresas

## Objetivo

Praticar conceitos fundamentais de banco de dados, incluindo:

- Modelagem conceitual
- Modelagem lógica
- Modelagem física
- Relacionamentos entre entidades
- Normalização de dados
- Estudos sobre RN (Regras de Negócio)

## Tecnologias e ferramentas
- BrModelo
- VScode
- MySQL Workbench
- MySQL Workbench
